4.说说filber架构

js和页面渲染线程有互斥性（或者说是阻塞），所以往往会出现js运算时间过长导致页面卡顿。因此React提出了filber（切片）的概念：把js运算左侧切片，js运行一段时间后调度器把线程控制权交给页面，页面渲染完成后再交回给js；有6个优先级标记任务的等级，原则是 willmount willreceiprops shouldComponentUpdate willupdate 等阶段的任务是可以被打断的；剩下的didmount didupdate 是不能被打断的。

第九-D3.js之动画

发表于 2021-06-27

动画

一 过渡动画

动画这一章比较有趣，同时d3的动画原理和css3的动画类似api也雷同。理解和掌握起来比较容易。

1、简单示例一：transition()、duration（）、delay() 生成一个从10到300的过渡动画

说明：调用了transition()后，返回的不再是选择器对象，而是一个过渡对象。得到过渡对象后，就可以调用动画相关的方法了

2、示例二：esse()动效效果

API 说明
selection.transition([name]) 过渡动画对象
transition.delay([value]) 延时
transition.duration([value]) 持续时间
transition.ease([value]) 缓动效果easing动画

二 过渡属性

transition.attrTween() 手动执行自定义动画，可以更精确的控制属性的变化

API 说明
selection.attrTween([name]) d 是被绑定的数据， i 是数据索引， a 是变化到最后的元素selection。 返回值也是一个函数，t的取值范围是 [0,1], 0 开始变化， 1 变化结束。
transition.styleTween([name]) 和attrTween类似，负责样式的过渡动画

transition.tween(name, factory) 自定义过渡动画（补间动画）；第一个参数貌似没什么用，第二个参数是动画逻辑。
