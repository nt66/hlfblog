发表于 2021-06-27

动画

过渡动画

动画这一章比较有趣，同时d3的动画原理和css3的动画类似api也雷同。理解和掌握起来比较容易。

1、简单示例一：transition()、duration（）、delay() 生成一个从10到300的过渡动画

说明：调用了transition()后，返回的不再是选择器对象，而是一个过渡对象。得到过渡对象后，就可以调用动画相关的方法了

2、示例二：esse()动效效果

API 说明
selection.transition([name]) 过渡动画对象
transition.delay([value]) 延时
transition.duration([value]) 持续时间
transition.ease([value]) 缓动效果easing动画
