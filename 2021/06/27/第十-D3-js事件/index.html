第十 D3.js事件

发表于 2021-06-27

事件

d3的事件可分为：鼠标事件|键盘事件|触屏事件|拖拽|缩放

鼠标事件比较简单，用on方法监听即可

API 说明
click 点击
mouseover 略了吧
mouseout 略了吧
mousemove 略了吧
mousedown 略了吧
mouseup 略了吧
dbclick 略了吧

键盘事件

keydown|keypress|keyup

API 说明
keydown 按下
keyup 弹起
keypress 特殊键盘按下

其他事件如 drag 等先略过
